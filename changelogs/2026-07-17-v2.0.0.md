# Changelog

All notable changes to NoScope are documented in this file.

## [v2.0.0] - 2026-07-17

### Highlights

- Rebuilt the visual editing experience around game-style player, staff, sponsor, and team cards with delayed hover-flip backs.
- Added a full team roster editor with main squad, bench squad, drag-and-drop reordering, swap-on-drop behavior, and roster-aware team views.
- Added custom list-view presets for Players, Staff, and Teams so dense database work can be split into focused views instead of one massive table.
- Added rating-driven sorting and leaderboard ranks, including gold, silver, and bronze badges for the top three players.
- Added bundled database Library support, CustomAssets access, and a built-in Guide panel for using the editor.
- Improved asset/logo matching, card performance, search responsiveness, and wide-table layout behavior.

### Added

- Added richer player search cards in the team roster editor, including portraits, nationality flags, age, roles, team logos, free-agent icons, and FACEIT badges.
- Added delayed player-card previews when hovering over roster-search results.
- Added FACEIT badges to player cards underneath the rating badge.
- Added multi-select support for card and list rows, including a Deselect all action when multiple entries are selected.
- Added sortable list-view headers with ascending and descending sorting.
- Added horizontal scrolling support for wide list views.
- Added game-style roster and bench cards to the team editor, including drag-to-reorder, swap-on-drop, and delayed hover-flip attribute backs.
- Added delayed hover-flip attribute backs to the main player-card grid.
- Added global leaderboard rank badges for players, including gold, silver, and bronze styling for ranks 1-3.
- Added header Guide, Library, and Assets buttons.
- Added an Assets panel with a Google Drive link to the CustomAssets folder.
- Added a Library panel for bundled databases, including counts, category icons, last-updated timestamps, download buttons, and EMDB.GG source links.
- Added five bundled Library databases: NoScope Default DB, Default EM2026 Database, Luca's GLOBAL UPDATED, 4's WorldUpdated, and Roy's EMDB Personal Edition.
- Added player grid sorting by alphabetical order, rating leaderboard, and team ERS.
- Added sponsor grid sorting by alphabetical order and tier.
- Added staff role-aware rating calculations and staff sorting by alphabetical order, rating, and team ERS.
- Added team sorting by alphabetical order and ERS.
- Added tournament sorting by alphabetical order and prize fund.
- Added custom player list-view presets for General, Gameplay Attributes, Mental Attributes, and Physical Attributes.
- Added custom staff list-view presets for General and role-specific attribute views.
- Added custom team list-view presets for General and Roster.
- Added roster backs to flipped team cards, showing main squad and bench players with portrait, country flag, and role.
- Added a custom Guide panel explaining loading, saving, views, sorting, editing, rosters, assets, filters, compare, and localhost behavior.
- Added dedicated team roster rows on flipped team cards, showing portrait, nickname, country flag, and primary role in a compact vertical roster layout.

### Changed

- Replaced the free-agent text badge with the bundled free-agent icon and tinted it to match the site accent styling.
- Updated list-view styling to better match the red editor theme and reduce the remaining blue accents.
- Stabilized list-view column widths so sorting indicators do not shift the table layout under the cursor.
- Updated Teams list view to show only `nick`, `name`, `country`, `earnings`, `rating`, `ers`, `academy`, `bgcolor`, and actions.
- Updated Tournaments list view to hide `name`, `description`, and `CUPID`.
- Updated generic country and nationality list cells to show country flags beside the text.
- Updated Teams card backgrounds to preview each team's `bgcolor`.
- Updated Teams list `bgcolor` cells to show a rounded color swatch beside the hex value.
- Replaced emoji toolbar icons with inline SVG icons.
- Moved Save and Export actions to the right side of the toolbar.
- Moved Guide, Library, and Assets into the header beside the NoScope branding.
- Changed player rank badges to use rating-based global leaderboard positions instead of row numbers.
- Changed non-medal player rank badges to use the red NoScope tone.
- Changed the bundled default database to `pzy_s_-_NoScope_MEGA_DB.emdb`.
- Renamed the Library source-link action to `EMDB.GG`.
- Removed the Library inactivity warning banner.
- Replaced text edit buttons in list views with the same pencil icon used on cards.
- Split player attribute list views into separate gameplay, mental, and physical views to reduce horizontal overflow.
- Updated player General list view to hide duplicated or low-value metadata columns such as `Nick`, `PR`, `Role3`, `Bio`, `ID`, `HLTV`, `Liquipedia`, `Faceit`, `Retired`, `FromFaceit`, and `Gender`.
- Updated sponsor card backs to show full descriptions while keeping descriptions off the front.
- Updated team card backs to show roster information instead of descriptions.
- Tuned player attribute list sizing so focused attribute views fit the panel more naturally.
- Updated `.gitignore` to keep local agent files and working-note documents out of published builds.
- Updated the release changelog and Discord announcement copy for the v2 launch.

### Fixed

- Fixed roster search results rendering blank when player search-card data was missing.
- Fixed player cards so clicking an already-selected card deselects it.
- Fixed Remove so it deletes every selected entry instead of only the most recently selected entry.
- Fixed Deselect all appearing when fewer than two entries were selected.
- Fixed wide list views pushing content past the viewport.
- Fixed `Legion` player cards incorrectly matching the `GamerLegion` logo by prioritizing exact team matches.
- Fixed player search and database loading slowdowns caused by recalculating leaderboard ranks for every rendered card.
- Fixed roster dragging so the whole card is picked up instead of leaving a ghost card behind.
- Fixed alphabetical descending behavior so alphabetical list sorting follows A-Z expectations by default.
- Fixed team roster list view so players after the first five display as bench even when roster-position fields are missing.
- Fixed team-card roster backs so players are displayed as unboxed horizontal rows stacked vertically by squad section.
