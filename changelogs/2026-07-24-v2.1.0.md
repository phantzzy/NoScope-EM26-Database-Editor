# Changelog

All notable public changes to NoScope are documented in this file.

## [v2.1.0] - 2026-07-24

### Highlights

- Added Mass Edit Beta for applying fixed values, randomized ranges, tier presets, and statuses to multiple selected players.
- Added Database Merge Beta for reviewing and selectively merging records, field updates, identity changes, and team transfers from a second `.emdb`.
- Added complete player Potential support across editing, cards, lists, filters, sorting, saves, and exports.
- Added save-time database validation for duplicate records and missing important fields.
- Added tournament location validation with eligible country and city combinations.
- Added a light theme while keeping dark mode as the default.

### Players

#### Added

- Added manual player Potential editing on a `1-20` scale.
- Added fractional `0-5` Potential stars to player cards, list views, and the player editor.
- Added player sorting and filtering by Potential.
- Added content creator badges and a dedicated Content Creator filter.
- Added FACEIT badges to player list views.
- Added birthday generation from a known age or a random age between `16` and `26`.
- Added a birthday clear action for quickly replacing an existing date.

#### Changed

- Potential now appears beside Rating in every player list preset.
- Players without a manually entered Potential continue to use an automatic value.
- Automatic Potential values are written into saved `.emdb` files and CSV exports.
- Saving a player with no birthday defaults the player to age `18`.
- Reorganized the Basic player filters and replaced the Gender text labels with sex-symbol icons.

### Teams and Rosters

#### Added

- Added player creation directly from the team roster editor.
- Added explicit main-squad and bench drop areas to roster drag and drop.
- Added support for placing players on the bench even when the main squad contains fewer than five players.

### Tournaments

#### Added

- Added validated country and city pickers using the tournament locations supported by the game.
- Added an OpenStreetMap preview for the selected tournament location.
- Added save validation that prevents unsupported country and city combinations.

#### Changed

- Tournament maps use a fixed, zoomed-out view so they do not interfere with editor scrolling.

### Mass Edit (Beta)

#### Added

- Added mass editing for multiple selected players.
- Added fixed-value and randomized-range editing for player attributes.
- Added quick `T1-T5` attribute presets.
- Added section-level field selection and bulk player status editing.
- Added validation and automatic Rating recalculation when applying changes.

### Database Merge (Beta)

#### Added

- Added database merging with the currently loaded database used as Database A and an uploaded `.emdb` used as Database B.
- Added selectable tables, player field groups, missing records, existing records, transfers, and compatible columns.
- Added a searchable review step where individual records and field changes can be approved or skipped.
- Added review support for new records, updates, team transfers, identity changes, and matching conflicts.
- Added clear side-by-side values for nickname, name, and other identity changes.
- Added a merge summary and one-step undo.
- Merges never delete records from Database A automatically.

### Validation and Saving

#### Added

- Added automatic validation before saving an `.emdb` file.
- Added a standalone Validate action for checking a database at any time.
- Added duplicate ID and name checks for Players, Staff, Teams, Tournaments, and Sponsors.
- Added checks for missing tables, columns, and important record fields.
- Added a grouped validation results window with affected record samples and a Save anyway option.
- Added a dynamic database loading screen with progress percentage and the current loading task.
- Added a Save or Exit prompt before replacing a database that contains unsaved changes.

### Countries and Flags

#### Added

- Added nationality options for the 193 UN member states, Kosovo, and additional game-compatible countries and territories.

#### Changed

- Replaced bundled country PNGs with consistent SVG flags from `flag-icons`.
- Removed unsupported nationality picker entries that are not valid in the game or editor.

### Interface

#### Added

- Added a dark and light theme switch beside the header tools.
- Added refreshed outline icons for Players, Staff, Teams, and Tournaments.
- Added a Changelog archive with expandable release notes inside NoScope.
- Added branded Steam and Discord header links for Esports Manager 2026.

#### Changed

- Dark mode remains the default whenever NoScope starts.
- Updated light-mode styling across cards, lists, editors, filters, rosters, modals, and navigation.
- Updated primary actions and roster section markers to consistently use the NoScope red styling.

### Guide and Documentation

#### Added

- Added eleven Guide screenshots covering the editor's primary workflows.
- Added in-app full-size screenshot previews.
- Added guidance for Potential, birthday generation, badges, team rosters, Mass Edit, Merge, validation, assets, and themes.
- Added archived release notes for `v2.0.0`.

#### Changed

- Reorganized the Guide around a four-step quick start and task-based feature sections.

### Fixed

- Fixed the From FACEIT filter not recognizing all supported stored boolean values.
- Fixed Compare not consistently using the selected players.
- Compare is now disabled without a selection and shows a custom warning when more than two players are selected.
- Fixed player cards flipping while the pointer or keyboard focus is on the Edit button.
